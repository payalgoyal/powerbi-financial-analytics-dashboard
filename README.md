# Financial Performance & Customer Insights Dashboard (Power BI)

## Overview

This project presents an interactive Power BI dashboard designed to analyze financial performance and customer insights across companies and departments.

The dashboard enables business users to monitor revenue, expenses, profitability, and customer contribution while identifying key performance trends.

## Dashboard Pages

### Executive Financial Overview

Provides a high-level snapshot of financial performance including revenue, expenses, profit, and profit margin trends.

### Profit & Loss Analysis

Analyzes departmental profit contributions using waterfall charts and highlights profit margin performance across departments.

### Expense Intelligence

Examines expense trends and departmental expense distribution to identify cost drivers.

### Revenue Insights

Analyzes revenue trends and distribution across departments and companies.

### Transaction Explorer

Provides detailed transaction-level insights.

### Customer Insights

Analyzes customer revenue contribution, profitability, and transaction behavior.

## Key Features

* Interactive slicers for company, department, and year
* KPI cards for financial monitoring
* Automated insight banners
* Trend analysis across months
* Department contribution analysis
* Customer profitability insights
* Reset filters functionality
* Tooltip pages for deeper analysis

## Tools Used

* Power BI
* DAX
* Data Modeling
* Data Visualization
